Run one job at a time, like a small terminal control plane.

This page is intentionally styled like a terminal window inspired by Ghostty. The backend is still the real Spring Boot service: jobs go into Postgres, the scheduler matches a small executor flavor, and short-lived worker pods run the command and report the result.
